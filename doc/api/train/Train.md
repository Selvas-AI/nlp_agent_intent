**Train**
----
  gets the trained model predictions for the input example

* **URL**

  /api/v1.0/apps/{app id}/train

* **Method:**

  `POST`

* **Data Params**

  * [FasttextWrapper](../../../FasttextWrapper.md)의 **Input Parameter format** 을 사용


* **Success Response:**
  
  * **Code:** 200 <br />
    **Content:** `Success`
        
 
* **Error Response:**

  * **Code:** 400 <br />
