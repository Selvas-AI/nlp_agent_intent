**get Trained Model Prediction**
----
  gets the trained model predictions for the input example

* **URL**

  /intent/predict

* **Method:**

  `GET`

*  **URL Params**

   **Required:**
 
   `example=[string]`

* **Success Response:**
  
  의도 및 점수를 반환한다.

  * **Code:** 200 <br />
        
 
* **Error Response:**

  * **Code:** 400 <br />
