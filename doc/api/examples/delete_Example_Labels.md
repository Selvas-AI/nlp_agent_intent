**delete Example Labels**
----
  This API should be called before calling the review API to get a list of ids for labeled utterances.

* **URL**

  /api/v1.0/apps/{app id}/examples/{example id}

* **Method:**

  `DELETE`


* **Success Response:**
  

  * **Code:** 200 <br />
    **Content:** `Success`
        
 
* **Error Response:**

  * **Code:** 400 <br />
